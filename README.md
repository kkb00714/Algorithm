※ 0529 => 파이썬 알고리즘 복습 

    1. 이진 탐색 알고리즘
    - 순차 탐색 : 리스트 안에 있는 특정한 데이터를 찾기 위해 앞에서부터 데이터를 하나씩 확인
    - 이진 탐색 : 정렬되어 있는 리스트에서 탐색 범위를 절반씩 좁혀가며 데이터를 탐색
        => 시작점, 중간점, 끝점을 이용하여 범위를 설정

        => 단계마다 탐색 범위를 2로 나누는 것과 동일하므로 연산 횟수는 log2 N 에 비례.
        
        => 따라서, 시간 복잡도는 O(log N)을 보장
